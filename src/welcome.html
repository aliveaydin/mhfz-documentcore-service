Welcome to mhfz-documentCore Service

Handles secure storage, retrieval, upload, delete, versioning, and export/sync of documents. Manages document objects, versions, and physical files, with support for multi-source, auditing, integrity checks, and external platform integration.

Available Routes

  • getDocument => GET /documents/:documentId
  • createDocument => POST /documents
  • updateDocument => PATCH /documents/:documentId
  • deleteDocument => DELETE /documents/:documentId
  • listDocuments => GET /documents
  • getDocumentVersion => GET /documentversions/:documentVersionId
  • createDocumentVersion => POST /documentversions
  • updateDocumentVersion => PATCH /documentversions/:documentVersionId
  • deleteDocumentVersion => DELETE /documentversions/:documentVersionId
  • listDocumentVersions => GET /documentversions
  • getFileObject => GET /fileobjects/:fileObjectId
  • createFileObject => POST /fileobjects
  • updateFileObject => PATCH /fileobjects/:fileObjectId
  • deleteFileObject => DELETE /fileobjects/:fileObjectId
  • listFileObjects => GET /fileobjects
